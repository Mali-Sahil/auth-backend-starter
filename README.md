# Auth Backend Starter

This is a beginner Node.js backend project created to learn authentication and core API concepts.

The project focuses on understanding how backend systems are structured, including authentication, validation, email verification, and database integration.

---

## Features

- User registration and login
- JWT-based authentication (access and refresh tokens)
- Email verification using Nodemailer and Mailgen
- Email testing using Mailtrap
- Input validation using express-validator
- MongoDB database connection
- Single User model
- Standard API response structure (ApiResponse and ApiError)
- Health check endpoint

---

## Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- JSON Web Tokens (JWT)
- Nodemailer
- Mailgen
- express-validator

---

## Environment Variables

This project uses environment variables for configuration.

An example file `.env.example` is included in the repository.  
Create a `.env` file in the root directory and add your own values.

---

## API Testing

A Postman collection is included in the repository at `postman/collections/`.

Import the collection into Postman to test all available API endpoints.

---

## Purpose

This project is built purely for learning purposes to understand backend authentication flows, API structure, validation, email workflows, and database connections.
It is not intended for production use.
